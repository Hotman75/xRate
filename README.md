# xRate

Instant currency conversion tool. Available as a standalone desktop app or integrated directly into Microsoft PowerToys, xRate provides fast and simple real-time exchange rates.

## ⚡ Quick Convert

xRate is available in the Command Palette for the fastest conversion tool possible.

* **Natural Syntax**: Type `100€$`, `100 USD EUR`, or just `100` to use your preferred currencies.
* **Auto-Recognition**: Supports currency symbols like $, €,...

## 🖥 Desktop Application

The desktop app provides a standalone tool.
